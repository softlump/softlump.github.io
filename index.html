Dojo Contest Submissions Per Day.

Collated this to help establish context for any given Dojo Contest. During an active contest this page will be updated daily. Submissions are not validated in order to be added, so any attempts to enter is counted even if key information is missing or the entry is obviously disqualified accoring to sumbission guidelines.

GMT +1, meaning that day 1 only covers about 5h, from 19:00 until 00:00. The last day represents 19h, as the threads close at 19:00 GMT+1.

This page is completely self-contained; styling, scripting and the data-set is all in this *.html file, so it can easily be saved for archival purposes.
